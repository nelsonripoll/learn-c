# Hello, World

## Preprocessor Directive

```
#include<stdio.h>
```

This is used to include the [stdio](http://www.cplusplus.com/reference/cstdio/) system header file.


```
int main() {
  printf("Hello, world!\n");
  return 0;
}
```

http://www.cplusplus.com/reference/cstdio/printf/


## Compile & Run

compile
```
gcc -o hello hello.c
```

run
```
./hello
```

output
```
Hello, world!
```
